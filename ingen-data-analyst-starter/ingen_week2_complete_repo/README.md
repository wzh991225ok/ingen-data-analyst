# inGen Dynamics Data Analyst Internship - Week 2 Deliverables

This repository package contains the Week 2 competitive intelligence deliverables for the Data Analyst Internship work plan.

## Week 2 scope
Week 2 converts the Week 1 robotics landscape into structured competitive intelligence. It selects 15 priority peers across five inGen-aligned verticals, normalizes competitor positioning, captures directional hiring signals, creates a patent-review tracker, and produces a competitive intelligence dossier.

## Folder structure
- `data/week02/competitor_profiles.xlsx` - detailed peer profiles and vertical summary
- `data/week02/peers_positioning.csv` - normalized competitor positioning dataset
- `data/week02/hiring_signals.xlsx` - hiring signal snapshot and summary metrics
- `data/week02/patents.xlsx` - patent review tracker and screening notes
- `reports/week02/competitive_intelligence_dossier.pdf` - 17-page CI dossier
- `reports/week02/executive_summary.pdf` - concise executive summary
- `reports/week02/status.md` - weekly status update
- `notebooks/week02_competitor_analysis.ipynb` - reproducible analysis scaffold

## Caution note
Hiring and patent counts change frequently. The included tables are structured research snapshots designed for an internship deliverable. Before using the outputs in an external investor or partner document, refresh the live career-page and patent-database counts and update retrieval dates.
